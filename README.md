# Como rodar o projeto

Pré requisitos: React Native e Android Studio 

```bash 
git clone https://github.com/leowingss/app-cronometro.git

# Entrar na pasta
cd app-cronometro

# Instalar dependências
npm install

# Executar o projeto
npx react-native run-android

```
